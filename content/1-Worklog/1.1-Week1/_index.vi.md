## Kết quả đạt được

* Hiểu được quy trình thực tập và môi trường làm việc tại Amazon Web Services Việt Nam.

* Nắm được các khái niệm cơ bản về Cloud Computing:

  * Infrastructure as a Service (IaaS)
  * Platform as a Service (PaaS)
  * Software as a Service (SaaS)
  * Khả năng mở rộng (Scalability)
  * Tính sẵn sàng (Availability)
  * Mô hình tính phí Pay-as-you-go

* Hiểu được cấu trúc hạ tầng toàn cầu của AWS:

  * AWS Region
  * Availability Zone
  * Edge Location

* Làm quen với AWS Management Console và cách sử dụng các dịch vụ trên nền tảng AWS.

* Tìm hiểu được chức năng của các dịch vụ nền tảng:

  * Amazon EC2
  * Amazon S3
  * Amazon VPC
  * AWS IAM
  * Amazon DynamoDB
  * AWS Lambda
  * Amazon API Gateway
  * Amazon Cognito

* Xác định được các dịch vụ AWS sẽ sử dụng trong đồ án xây dựng hệ thống Backend Game Multiplayer.

* Hoàn thành kế hoạch nghiên cứu cho giai đoạn triển khai hệ thống.
