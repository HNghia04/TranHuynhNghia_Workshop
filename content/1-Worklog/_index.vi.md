Chương trình thực tập được thực hiện trong khoảng 3 tháng, từ ngày 05/05/2026 đến ngày 29/07/2026, với tổng cộng 12 tuần. Nội dung công việc tập trung vào việc nghiên cứu, thiết kế và triển khai kiến trúc Backend cho game multiplayer trên nền tảng Amazon Web Services.

Các nội dung thực hiện theo từng tuần gồm:

**Tuần 1:** [Làm quen với môi trường thực tập, Cloud Computing và các dịch vụ AWS cơ bản](1.1-week1/)

**Tuần 2:** [Nghiên cứu Amazon Cognito, API Gateway, AWS Lambda và Amazon DynamoDB](1.2-week2/)

**Tuần 3:** [Tìm hiểu Amazon EC2, Node.js và giao tiếp thời gian thực bằng WebSocket](1.3-week3/)

**Tuần 4:** [Nghiên cứu kiến trúc game multiplayer, Matchmaking và đồng bộ dữ liệu thời gian thực](1.4-week4/)

**Tuần 5:** [Phân tích yêu cầu đề tài, thiết kế kiến trúc hệ thống và lập kế hoạch triển khai](1.5-week5/)

**Tuần 6:** [Hoàn thiện kiến trúc, mô hình dữ liệu và luồng tích hợp giữa các dịch vụ AWS](1.6-week6/)

**Tuần 7:** [Triển khai hạ tầng AWS, cấu hình EC2, Security Group và xây dựng WebSocket Server](1.7-week7/)

**Tuần 8:** [Xây dựng chức năng xác thực, API Serverless, Matchmaker Lambda và DynamoDB](1.8-week8/)

**Tuần 9:** [Tích hợp Game Client với Matchmaking và đồng bộ trạng thái qua WebSocket](1.9-week9/)

**Tuần 10:** [Kiểm thử toàn bộ hệ thống, sửa lỗi đồng bộ và tối ưu hiệu năng](1.10-week10/)

**Tuần 11:** [Hoàn thiện chức năng, xử lý dữ liệu sau trận đấu và hoàn thiện tài liệu kỹ thuật](1.11-week11/)

**Tuần 12:** [Hoàn thiện báo cáo thực tập, đánh giá kết quả và chuẩn bị sản phẩm](1.12-week12/)
